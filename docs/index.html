How it works
Three pieces. Total transparency.
1

Describe your project in a config

Copy app/configs/example.ts, list your real source files as blocks with their roles and imports, and register it. The AI assistant can generate this for you in one prompt.

2

Expose two read-only endpoints on your backend

GET /metrics returns live edge counts, byte totals, and per-block health. GET /logs streams recent log lines. Both are in-memory only — no database required.

3

Open the dashboard

Run npm start in the CodeXray folder. Open localhost:3001/?project=yourapp. The map renders instantly; traffic lights up as your app runs.

Quick start
Running in under a minute

Clone alongside your project and start the dashboard:

git clone https://github.com/amitfounderspace/CodeXray.git
cd CodeXray
npm start

Requires Node.js LTS. Then open localhost:3001 in your browser. Not a developer? Double-click start.command — it handles everything.
